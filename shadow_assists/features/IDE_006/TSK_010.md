## 1. TL;DR

현재 cold path C++ kernel 이 OMP `parallel for` 로 *단일 team* 의 N threads (TSK_004 의 partition cpulist 크기). prod 56 cores × 2 socket 에서 한 worker 가 14 cores 만 받으면 (TP=8), kernel 이 14 threads 만 사용. 나머지 cores 는 다른 worker 가 사용 중이지만, *batch 안의 다른 sub-batch* 가 별도 team 으로 동시 진행하면 *팀 간 병렬* 가능.
