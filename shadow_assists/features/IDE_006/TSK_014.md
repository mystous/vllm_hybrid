## 1. TL;DR

NEO 의 *3 큐* (waiting / GPU running / CPU running) 와 *6 단계 batch 결정* (swap-out / swap-in / prefill 분류 / mode selection / launch) 을 vLLM scheduler 에 적재한다. `Scheduler.schedule()` 의 반환을 *두 SubBatch + swap list* 로 확장하고, 별도 `mode_selector.py` 에 NEO 의 `_decide_mode_and_gen_batch` 를 port 하여 *pipelined vs sequential* 을 PerfPredictor (`TSK_017`) 의 입력으로 결정.

### 2.2 · vLLM 기존 추상과의 정합

| vLLM 기존 | NEO 식 적용 후 |
|---|---|
| `Scheduler.schedule()` → `SchedulerOutput` | `Scheduler.schedule()` → `TwoSubBatchSchedulerOutput` (= 두 SubBatch + swap list) |
| `running` (deque) | `gpu_decoding_q` (list) + `cpu_decoding_q` (deque) — 두 큐로 분할 |
| `waiting` (deque) | 그대로 (단 prefill 분류 시 GPU/CPU 결정 추가) |
| `_schedule_running` budget | `ScheduleBudget` (max_batch_size + max_tokens_in_batch) |

## 4. 알고리즘 reference

자세한 NEO 알고리즘은 [`NEO_code_deepdive.md`](NEO_code_deepdive.md) §3 (Scheduler 6 단계 + mode selection 5 단계 + `_get_remains` 시간 매칭).

핵심 의사 코드 (vLLM 위 재구현):

```python
# vllm/v1/core/sched/scheduler.py (확장)
class Scheduler:
    def schedule(self) -> TwoSubBatchSchedulerOutput:
        budget = ScheduleBudget(self.max_batch_size, self.max_tokens_in_batch)
        swap_out_threshold = self.num_gpu_blocks
        swap_in_threshold = round(swap_out_threshold * 0.95)

        # Step 1: GPU decoding budget reserve
        gpu_block_needed = sum(req.num_blocks for req in self.gpu_decoding_q)
        budget -= len(self.gpu_decoding_q)

        # Step 2: Swap-out
        swp_out = []
        while budget.overspent or gpu_block_needed > swap_out_threshold:
            victim = self.gpu_decoding_q.pop()
            self.cpu_decoding_q.appendleft(victim)
            swp_out.append(victim)
            ...

        # Step 3: Swap-in
        swp_in = []
        while self.cpu_decoding_q and gpu_block_needed + need <= swap_in_threshold:
            ...
        assert not swp_out or not swp_in

        # Step 4: 새 prefill 분류 (GPU 빈 자리 우선 → CPU)
        pref_to_gpu, pref_to_cpu = [], []
        for cand in self.waiting_q:
            if not pref_to_cpu and gpu_block_needed + need <= self.num_gpu_blocks:
                pref_to_gpu.append(cand)
            else:
                pref_to_cpu.append(cand)

        # Step 5: pipelined vs sequential 결정
        batches = self.mode_selector.decide(
            pref_to_gpu, pref_to_cpu,
            self.gpu_decoding_q, self.cpu_decoding_q,
            self.predictor, budget
        )

        # Step 6: 실제 prefill launch
        ...

        return TwoSubBatchSchedulerOutput(batches, swp_out, swp_in)
```
