## 1. TL;DR

NEO мқҳ *table-based linear interpolation* perf predictor вҖ” `linr_T(S)` (linear) / `pref_T(S)` (prefilling) / `gdec_T(N)` (GPU decode) / `cdec_T(S, N)` (CPU decode bilinear) / `lnch_T()` (kernel launch overhead) мқҳ 5 мў…лҘҳ prediction. мқҙкІҢ TSK_014 мқҳ mode selection (pipelined vs sequential) мқҳ кІ°м • мһ…л Ҙ.

`TablePerfPredictor` лҠ” startup мӢң `ModelProfiler` к°Җ *к°Ғ (S, N) м җм—җм„ң actual forward мёЎм •* н•ҳм—¬ table мұ„мӣҖ. мқҙнӣ„ scheduling мқҳ л§Ө iteration м—җм„ң O(1) lookup + linear interpolation.

### 2.2 В· Profile run мқҳ cost / мҳҒн–Ҙ

NEO мқҳ profile run мқҖ *к°Ғ (S, N) м җм—җм„ң forward мёЎм •* вҖ” N м җмқҙ ~мҲҳмӢӯ к°ң. prod нҷҳкІҪм—җм„ң startup overhead мёЎм • мҳҒм—ӯ:

- к°Ғ мёЎм • м җ = 1 forward step (~мҲҳ ms ~ мҲҳмӢӯ ms)
- мҙқ мёЎм • м җ вүҲ `len(linr_S_list)` + `len(pref_S_list)` + `len(gdec_N_list)` + `len(cdec_S_list Г— cdec_N_list)` вүҲ мҲҳл°ұ
- мҳҲмғҒ startup overhead: < 60s (TST_017 мқҳ acceptance)

### 3.1 В· 1.5/1.6 м Ғмһ¬ plan вҖ” engine мёЎ measure_fn

NEO мқҳ `_run_test_case` лҠ” artificial SubBatch л§Ңл“  нӣ„ `executor.do_one_iteration` нҳём¶ң. vLLM мң„м—җм„ң лҸҷмқј мҳҒм—ӯ:

```python
# vllm/v1/metrics/profiler.py мқҳ engine-side adapter
def make_measure_fn(model_runner, vllm_config) -> Callable[[str, int, int], float]:
    def measure(kind: str, p1: int, p2: int = 0) -> float:
        # kind вҲҲ {"linr", "pref", "gdec", "cdec", "lnch"}
        # p1 = S (batch width) / p2 = N (kv tokens)
        # 1) artificial SubBatch л№Ңл“ң вҖ” kind лі„лЎң add_pref/add_gdec/add_cdec
        # 2) model_runner._dummy_run(...) к°ҷмқҖ кё°мЎҙ dummy run мқён”„лқј reuse
        # 3) torch.cuda.synchronize() м „нӣ„мқҳ wall-time мёЎм • (ms)
        ...
    return measure
```

vLLM мқҳ `gpu_model_runner.py` м—җлҠ” мқҙлҜё dummy run мқён”„лқјк°Җ мһҲмқҢ (`_dummy_run`, `dummy_run` л“ұ). NEO profile мҳҒм—ӯмқҳ м§Ғм ‘ reuse к°ҖлҠҘ.

### 3.3 В· 1.7 caching м „лһө

- лҸҷмқј `(model_name, gpu_arch, dtype, tp_size, max_num_seqs, block_size)` tuple м—җ лҢҖн•ң table мқҖ *deterministic* (workload-independent measurement).

class PerfPredictor:
    def get_linr_T(self, S: int) -> float: ...   # linear (FFN/proj) мӢңк°„

    def get_cdec_T(self, S: int, N: int) -> float: ...  # CPU decode (bilinear)
    def get_lnch_T(self) -> float: ...           # launch overhead

class TablePerfPredictor(PerfPredictor):
    def __init__(self, engine_config):
        # x к°’ list вҖ” 2 мқҳ кұ°л“ӯм ңкіұ + ліҙк°• м җ
        self.linr_S_list = list(range(1, 512)) + [2**i for i in range(9, ...)]
        # ... лӢӨлҘё list л“ӨлҸ„ м •мқҳ

        # Lower-bound index лҜёлҰ¬ кі„мӮ° (O(1) lookup)
        self.linr_S_lb_idx = self._get_lb_idx_list(self.linr_S_list)
        # ...

    def _interp(self, x, x0, x1, y0, y1):
        return y0 + (y1 - y0) * (x - x0) / (x1 - x0)

    def _interp_1d(self, x, xs, ys, lb_idx):
        if x == 0: return 0.0
        idx = lb_idx[x]
        if idx == 0 or x == xs[idx]: return ys[idx]
        return self._interp(x, xs[idx-1], xs[idx], ys[idx-1], ys[idx])

        ...

`ModelProfiler` мқҳ startup мёЎм • ([`NEO_code_deepdive.md`](NEO_code_deepdive.md) В§6.4):

```python
# vllm/v1/metrics/profiler.py
class ModelProfiler:
    def init_profile_tables(self, block_manager):
        # 4 мў…лҘҳ table мұ„мҡ°кё°
        self.pp.linr_S_list, self.pp.linr_T_list = self._profile_linr(...)
        self.pp.pref_S_list, self.pp.pref_T_list = self._profile_pref(...)
        self.pp.gdec_N_list, self.pp.gdec_T_list = self._profile_gdec(...)
        self.pp.cdec_S_list, self.pp.cdec_N_lists, self.pp.cdec_T_lists = \
            self._profile_cdec(...)

    def _run_test_case(pref_lens, gdec_lens, cdec_lens, nwarmup=2, nrepeat=3):
        # artificial test case лЎң SubBatch л§Ңл“  нӣ„ actual forward мӢӨн–ү
        nbatches = len(pref_lens)  # 1 (sequential) лҳҗлҠ” 2 (pipelined)
        batches = [SubBatch() for _ in range(nbatches)]
        # ... add_pref / add_gdec / add_cdec
        args = self.bm.prepare(batches, [], [])

        for i in range(-nwarmup, nrepeat):
            if i == 0: self.executor.turn_on_perf_monitor()
            self.executor.do_one_iteration(batches, *args)

        return self.executor.turn_off_perf_monitor_and_flush_results()
```

н•өмӢ¬ вҖ” л‘җ ліҖмў…:
- `_run_test_case_seq` = лӢЁмқј SubBatch (sequential мёЎм •)
- `_run_test_case_pip_same` = л‘җ лҸҷмқј SubBatch (pipelined мёЎм •)

мҙқ мёЎм • м җ вүҲ мҲҳл°ұ, nwarmup=2 + nrepeat=3 вҶ’ startup мӢңк°„ acceptance < 60s (TST_017).
