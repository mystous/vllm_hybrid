## 1. TL;DR

vLLM мқҳ *mirror* м •мұ… (cold blocks к°Җ GPU + CPU м–‘мӘҪ мһ”лҘҳ) мқ„ NEO мқҳ *request лӢЁмң„ exclusive ownership* мңјлЎң ліҖкІҪ. н•ң request мқҳ KV лҠ” GPU лҳҗлҠ” CPU н•ң мӘҪм—җ *exclusive* лЎң мЎҙмһ¬. swap-in/out мқҖ *atomic* вҖ” source м—җм„ң free + dest м—җ alloc.

мқҙ м •мұ… ліҖкІҪмқҙ NEO мқҳ *batch нҷ•мһҘ* мқ„ к°ҖлҠҘн•ҳкІҢ н•ҳлҠ” fundamental мҳҒм—ӯ. mirror лқјл©ҙ GPU кіөк°„мқҙ cold blocks лЎң м җмң лҗҳм–ҙ batch size лӘ» нӮӨмӣҖ.

mirror policy лҠ” *config flag лЎң м„ нғқ к°ҖлҠҘ* вҖ” кё°ліёмқҖ mirror (кё°мЎҙ user л¬ҙнҡҢк·Җ), `--kv-cache-policy=exclusive` лЎң NEO мӢқ нҷңм„ұ. TSK_014 мқҳ scheduler к°Җ *exclusive* лҘј к°Җм •н•ҳлҜҖлЎң NEO мӢқ м Ғмһ¬ мӢң н•Ёк»ҳ нҷңм„ұ.

## 4. м•Ңкі лҰ¬мҰҳ reference

[`NEO_code_deepdive.md`](NEO_code_deepdive.md) В§5.

н•өмӢ¬ invariant вҖ” *exclusive*:

```python
# м–ҙлҠҗ мӢңм җмқҙл“ :
for req in all_requests:
    has_gpu = req in gpu_block_manager.allocated
    has_cpu = req in cpu_block_manager.allocated
    assert has_gpu ^ has_cpu  # XOR вҖ” м •нҷ•нһҲ н•ң мӘҪ
```

`_initiate_swap` мқҳ atomic:

```python
def _initiate_swap(reqs, is_swap_out):
    src = gpu_block_manager if is_swap_out else cpu_block_manager
    dst = cpu_block_manager if is_swap_out else gpu_block_manager
    src_pids = src.free(reqs)              # source мқҳ block нҡҢмҲҳ
    dst_vids, dst_pids = dst.alloc(reqs)   # dest м—җ alloc
    return src_pids, dst_vids, dst_pids
```

вҶ’ swap мқҳ *physical transfer* (PCIe) лҠ” лі„лҸ„ worker thread м—җм„ң. ліё TSK мқҳ мҳҒм—ӯмқҖ *block table мқҳ ownership к°ұмӢ  atomic*.
