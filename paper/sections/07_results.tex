\subsection{핵심 결과 1: 단일 기법의 유의 임계점 도달 실패}

Table~\ref{tbl:vs-baseline}은 24개 후보 기법의 baseline 대비 효과를 1-run 측정값 큰 순으로 정렬한 결과이다.
주요 관찰은 다음과 같다.

\textbf{관찰 1.} 단일 기법 중 \emph{branchy work $\times$ 100\,ms} (분류군 E)만이 유일하게 임계점 +5\,\%에 도달하였다 ($\Delta = +5.28\,\%$).
이는 \S\ref{subsec:theorem2}의 Theorem~\ref{thm:alignment}이 예측한 정렬 이득의 직접 검증이다.

\textbf{관찰 2.} 분류군 A (직접 커널 대체) 7개 기법 모두 임계점에 도달하지 못했으며, 6개 기법은 net-negative이다.
이는 \S\ref{sec:problem}의 \emph{원인 1: critical path 침범}을 직접 검증한다.
특히 ``4-method Jacobi router''는 $-94.2\,\%$의 catastrophic 결과를 보였는데, 이는 LM-head 연산이 1{,}810\,ms $\times$ 165 chat 요청에 직렬화되어 큐 정체를 발생시킨 결과이다.

\textbf{관찰 3.} 분류군 C (sub-step CPU 스케줄링) 3개 기법 모두 net-negative이며, 특히 task-pool dummy fill 기법은 spec-decode backend에서 $-14$\,\%부터 $-20$\,\%의 catastrophic 결과를 보였다.
이는 task-pool의 lock 경쟁이 step 내부 sub-phase 경계의 critical path를 stall 시킨 결과이다.

\subsection{핵심 결과 2: 교차 도메인 중첩 정리의 경험적 검증}

\input{tables/tbl_superposition}

Table~\ref{tbl:superposition}는 Theorem~\ref{thm:superposition}의 경험적 검증 결과이다.

\textbf{Cross-domain pair (NUMA pin $\oplus$ auxiliary work)}: 두 단일 기법의 합은 $+1.54\,\% + +1.84\,\% = +3.38\,\%$로 예측되었고, 실제 측정값은 $+2.83\,\%$로 residual은 $-0.55$\,pp에 머문다.
이는 near-linear superposition의 강한 신호이다.

\textbf{Same-domain stack (4-OS-lever)}: 단일 기법 합 $+1.15\,\%$이 예측되었으나 실측은 $-0.03\,\%$로 residual $-1.18$\,pp의 destructive interference가 발생한다.
3개 이상의 동일 도메인 stack은 $-3.93$\,pp까지 발산하여 superlinear destructive를 확인한다.

이 결과는 \algname 의 Stage 2 (\textsc{Chord})의 design 결정---\emph{서로 다른 도메인에 속하는 두 기법만 쌍으로 stacking}---을 정당화한다.

\subsection{핵심 결과 3: multi-run 분산 분석}

\input{tables/tbl_multirun}

Table~\ref{tbl:multirun}는 핵심 후보 3개 기법의 multi-run 측정 결과이다.

\textbf{NUMA pin은 robust positive}: 1-run $+1.54\,\%$가 3-run mean $+2.24\,\%$, warm-only $+3.13\,\%$로 일관되게 양의 효과를 유지한다.
이는 hardware-level NUMA distance 비대칭(local 10 vs. remote 21)이 매 측정에 일관된 영향을 미치기 때문이다.

\textbf{tokenize 20\,ms는 retract}: 1-run $+1.66\,\%$의 신호가 3-run mean에서 \emph{부호 반전}하여 $-5.96\,\%$로 변한다.
이는 1-run 측정의 신뢰도 한계를 직접 보여주며, magnitude $|\Delta| < 3\,\%$의 단일 측정은 결정적 신호로 채택할 수 없음을 시사한다.
본 발견은 negative result 측정의 best practice에 대한 일반적 함의를 제공한다.

\subsection{최종 통합: \algname 의 효과}

위 hyperparameter로 구성된 \algname 의 final configuration --- $L_{\text{primary}} = $ NUMA pin, $L_{\text{secondary}} = $ branchy auxiliary work, $\tcpu = 100$\,ms --- 의 측정 결과는 다음과 같다.

\begin{itemize}
  \item 3-mix 평균 lever-only $\Delta = +5.28\,\%$ (Theorem~\ref{thm:alignment}의 예측값과 정합)
  \item 단일 cell (balanced workload) 최대 $\Delta = +12.04\,\%$
  \item vanilla 대비 vs-vanilla full delta = $+174.96\,\%$ (적응형 라우팅 기준 효과 + \algname 효과 합산)
  \item Wall-clock 단축: $-5.40\,\%$ (3-mix 평균), $-10.74\,\%$ (balanced)
\end{itemize}

24개 단일 후보 기법이 모두 미달한 유의 임계점 +5\,\%를, \algname 의 단일 통합 구성이 도달했다는 점이 본 논문의 핵심 검증 결과이다.
